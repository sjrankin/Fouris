GameUINotificationProtocol

protocol GameUINotificationProtocol : AnyObject

Provides notifications of events from the abstract game engine to the UI. The game engine knows very little about the UI (even though it’s written in Swift), as there are two UIs (macOS and iOS) and three interfaces per UI available.

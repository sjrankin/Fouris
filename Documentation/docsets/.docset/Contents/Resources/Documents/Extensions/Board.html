Board

Extensions related to the piece ID map.

  • InitializePieceIDMap()

    Initialize the piece map.

  • IDtoPiece(_:)

    Return the piece type given an ID of the piece.

  • AddID(_:ForPiece:)

    Add a piece’s ID to the piece ID map.

  • ChangeType(For:ToType:)

    Changes the type of the piece an ID points to.

  • ClearPieceMap()

    Clear the piece ID map. Reload with the invariant pieces.

  • PieceMap

    Holds the ID to piece type map.

  • RemoveID(_:)

    Remove the specified ID from the piece ID map.

  • IsEmptyType(_:)

    Determines if the piece type pointed to by ID is empty in the context of moving the user’s piece (eg, can the user move a piece into the location?).

  • IsOccupiedType(_:)

    Determines if the piece type pointed to by ID is occupied in the context of moving the user’s piece (eg, the location is blocked so the user cannot move there).

  • IsValidNeighborType(_:)

    Determines if the piece type pointed to by ID is a valid neighbor. Used by the AI.

  • IsSpecialType(_:)

    Determines if the piece type pointed to by ID is a special type (eg, action or danger button).

  • StaticID(For:)

    Returns a static ID for the given piece type. Not all piece types have associated static IDs. (Static means unchanging for the duration of the instance.)

    Note

    Note:

    • Piece types with static IDs are: .Visible, .Background, .Bucket, .InvisibleBucket.
    • If a piece type is passed that does not have a static ID, a fatal error is generated.
  • UniqueIDs(BoardMap:)

    Returns a set of unique IDs found in the board.

  • UniqueIDCount(BoardMap:)

    Returns the number of unique IDs in the passed board.

    Note

    This is merely a thin wrapper around UniqueIDs.

  • RemoveUnusedIDs(BoardMap:ButNotThese:)

    Remove IDs from the piece ID map that are no longer in the passed board.

  • AddPieceAttributes(_:_:)

    Add attributes from a piece.

  • GetAttributes(For:)

    Get attributes for the piece whose ID is passed.

  • RemoveAttributes(For:)

    Remove the attributes (because the piece no longer exists) for the piece with the passed ID.

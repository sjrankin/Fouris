  • PhoneType()

    Returns the type of phone we’re running on. If not running on a phone, the type returned will reflect that.

    Declaration

    Swift

    static func PhoneType() -> PhoneTypes

    Return Value

    Phone type if on a phone. If not on a phone, either .Simulator or .NotOnPhone will be returned.

  • OnPlusPhone()

    Determines if the current phone is on a plus-sized phone.

    Declaration

    Swift

    static func OnPlusPhone() -> Bool

    Return Value

    True if on a plus (large) phone, false if not (or if on the simulator or non-phone device).

Board

class Board

Maintains the board for the game. The board makes heavy use of the Map instance to maintain the location of various blocks on the board. The board sits between the in-play piece(s) and the game instance.

  • PieceInBounds(_:)

    Determines if a piece is fully in the bucket or not. Outside the bucket is determined by having at least one piece having a vertical value less than (eg, higher than) the top of the bucket.

    • Notes:

      • Call only after the piece starts to freeze. This function calls a Map function that updates the score.
      • This function is used to determine game over conditions. If a piece is frozen into place with at least one block over the top of the bucket, game over conditions are met.

    Declaration

    Swift

    func PieceInBounds(_ ThePiece: Piece) -> Bool

    Parameters

    ThePiece

    The piece to check for in-boundedness.

    Return Value

    True if the passed piece is fully in the bucket, false if not.

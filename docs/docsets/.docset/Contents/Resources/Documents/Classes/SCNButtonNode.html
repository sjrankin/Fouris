    Note

    • The total time from start to finish is ResetDuration + Delay.

    ResetColor

    The color to change to over a period of time. Assumed to be (not not necessary to be) the same as the HighlightColor.

    Amount of time for the color to change from NewColor to ResetColor, in seconds.

ColorPickerCode

class ColorPickerCode : UIViewController, ColorPickerProtocol

Runs the color picker UI. This dialog allows the user to manually create colors via sliders. There is one slider for each color channel in a colorspace. Most color spaces have three channels, but four channels are supported. Supported color spaces are: RGB, HSB, YUV, and CMYK.

    Get or set the shape name.

    Note

    Due to the way Swift interacts with Objective-C runtimes and the Interface Builder, we can’t use enums for @IBInspectable properties. To get around that, the caller can set this property to a string value. If the string value is recognized as one of the raw values for the TileShapes3D enum, it will be converted and the shape will change as appropriate. If the string contains an unrecognizable value, no change will occur.

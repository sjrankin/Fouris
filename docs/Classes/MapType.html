MapType

class MapType : CustomStringConvertible

Encapsulates a map and ancillary data for the game board.

Note

  • There are two two main layers to the map: The first layer is the piece type layer which identifies the type of piece in a give location in the map, such as bucket, visible (empty), retired piece, active piece, and the like. The second layer is the block map which is a map of identifies of which individual block occupies which location.

ConstrainedQueue

class ConstrainedQueue<T>

Implements a simple generic queue with a limit on the number of items that can be stored.

Note

When more items are inserted into the queue than it can hold, the oldest items are removed and discarded.

    Get or set the maximum number of items. If the caller sets this value small than the number of items in the current queue, the excess will be discarded. Setting this value to a larger number than the number of items in the queue will have no immediate effect.

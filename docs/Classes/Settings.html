    Returns the contents of the most recently used colors.

    Note

    • The number of items in the color list is limited by the value returned by GetMostRecentlyUsedColorListCapacity when the list was last written.
    • The format of the returned string is Hex Value,Hex Value, where Hex Value is a hex color value. The delimiter between values is a comma (**,**).

    Declaration

    Swift

    public static func GetMostRecentlyUsedColorList() -> String

    Return Value

    The contents of the most recently used color list. If the user has not set any colors or on first execution after installation, an empty string will be returned.

  • SetMostRecentlyUsedColorList(NewValue:)

    Set the contents of the most recently used colors list to the user defaults.

    Note

    • The number of items should not exceed more than the value returned by GetMostRecentlyUsedColorListCapacity. However, this function does not enforce that limit.
    • The format of the passed string is Hex Value,Hex Value, where Hex Value is a hex color value. The delimiter between values is a comma (**,**).

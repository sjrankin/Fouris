ColorNamePickerCode

class ColorNamePickerCode : UIViewController, UIPickerViewDelegate, UIPickerViewDataSource, ColorPickerProtocol

Runs the UI for the color name picker. Color names are in a shallow heirerachy of a set of groups - each group has its own set of colors. The user can select the group, then a color from the group.
